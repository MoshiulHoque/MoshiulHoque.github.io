ACTIVITY: DAY 4

JULY 18, 2018 by Moshiul Hoque

Today we were learnt about the few important features of Github, 3D modeling with NX11, Onshape, and Operation of CNC machine in different sessions.

Session 1: Publishing webpage with Github (10:00 ~ 11:00)

We learnt about the several importance features of Github such as installing Github software, setting up of it, Create a repository, Clone the repository, Push it, Enabling GitHub Pages to publish your site from adding theme to the existing webpage, changing contents of the html file, writing simple html program, etc. After learning the topics, we published our designed file into the web.

Tea Break: 11:00 ~ 11: 15

Session 2: 11:15 ~ 12:00: Html

Session 3: (12:00 ~ 1:15) NX11

We learnt the basic concept of 3D modeling using NX11.0 software. We made the sample sketch design and save it. We use setting the workspace, setting the origin, line drawing, circle drawing, spline curve, drawing, rectangle, etc. After learning the basic design concepts and tools, the instructor gives us few sample design to draw. We draw these designs competently.

Session 4: 2:00~4:00 OnShape

This session we cover the Onshape software, a free-to-use, powerful, professional 3D CAD package that’s cloud and browser based. Setting Up an Account.

Tea Break: 4:00 ~ 4:15

Session 5: 4:15 ~ 6:00: Rdworks.
